# 💊 Pharma Inventory Dashboard

## 📌 Overview

This project presents a real-time pharmaceutical inventory dashboard designed to monitor stock levels, expiry risks, and critical alerts.

It helps ensure efficient inventory management and reduces wastage in healthcare supply chains.

📍 Use Case: Applicable to pharmaceutical supply chains and hospital inventory systems

---

➡️ The dashboard tracks:

* Stock levels
* Low stock alerts
* Expiry risks
* Real-time inventory status
